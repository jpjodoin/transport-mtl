----------------------------Français---------------------------
Ce dossier contient le projet Eclipse Android de Transport Montréal. Il y a également certains fichiers de configuration afin que celui-ci fonctionne sur Blackberry 10. Pour faire fonctionner l'application, il suffit d'importer les deux dossier dans éclipse, c'est à dire la librairie AmbilWarna qui permet de choisir une couleur pour les groupes dans l'application ainsi que le dossier transportmtl qui le dossier de l'application en tant que tel.

transportmtl_legacy est la version qui supporte Android 1.5, 1.6
transportmtl est la version qui supporte Android 2.1 et plus.

----------------------------English----------------------------
This folder contains the Android project for Eclipse of Transport Montréal. Some configuration file for BB10 are also included. To make the application work, you will need to import the project transportmtl in Eclipse. You will also need to import the library AmbilWarna that was copied from here: (http://code.google.com/p/android-color-picker/). This library provides a color picker.
